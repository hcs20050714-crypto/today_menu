# 오늘 뭐 먹지? 🍴

점심과 회식 메뉴를 랜덤으로 추천해주는 웹페이지입니다.

## 기능

- 📍 지역명 입력으로 맞춤 검색
- 🍱 점심 메뉴 20개 중 랜덤 추천
- 🍻 회식 메뉴 20개 중 랜덤 추천
- ✨ 1초간 메뉴가 바뀌는 애니메이션 효과
- 🎉 결과 표시 시 색종이(Confetti) 효과
- 🗺️ 네이버 지도 연동으로 주변 식당 검색

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 지역명 입력 (예: 나성동, 강남역)
3. "점심 메뉴" 또는 "회식 메뉴" 버튼 클릭
4. 추천된 메뉴 확인
5. "내 주변 이 메뉴 식당 보기" 버튼으로 네이버 지도에서 검색

## 기술 스택

- HTML5
- JavaScript (Vanilla)
- Tailwind CSS
- Canvas Confetti
